CSS Selectors Table

CSS selectors & Functions

Selector

Function
<link> The <link> element links HTML documents to external resources, such as CSS files.
Class and ID selectors Classes (denoted with a period) can be used for multiple elements. ID selectors ( denoted by a #) can only be used for 1 element.
CSS Type Selectors Modifies all types of the given element. They are usually the element without the "< >" around them. For example h2 will affect the CSS properties of any < h2 > in the HTML file.
